## Blogecho for Typecho 概述
- 作者：JackieLiu
- 版本：v1.0
- 更新日期：2016.11.28

## 特性
 1. 集成代码高亮
 2. 完美兼容 MarkDown 语法
 3. 响应式布局
 4. 随机更换透明背景壁纸

## 使用方法
1. 点击 download zip 下载最新源码，解压，将其中文件夹重命名为 Blogecho 并上传至博客的 /usr/themes 目录下
2. 可将img文件夹下 background 背景图片替换为自习喜欢的，logo.jpg 替换为自己的 logo 头像
3. 在主题外观设置页面，设置相应选项
4. 在博客后台点击启用
